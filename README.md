# Did Slavery Impede the Growth of American Capitalism?

Joseph Francis, *Did Slavery Impede the Growth of American Capitalism?
Two Failed Natural Experiments* — working paper, current version
**v6.1** (July 5, 2026).

## Layout

- **`Francis_Farm_values.pdf`** — the current version of the paper.
- **`replication/`** — the replication package for the current version:
  a single `Master.R` pipeline that downloads the census data, builds the
  database, and produces every table, figure, and map in the paper. See
  `replication/README.md` for requirements and instructions.
- **`archive/`** — superseded versions of the paper. Versions before
  v6.0 were distributed as PDFs only; `archive/v6.0/` contains the full
  v6.0 release, including its replication package, as published here on
  May 25, 2026.
- **`CHANGELOG.md`** — what changed in each version.
